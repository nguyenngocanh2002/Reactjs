vòng lặp trong cuộc sống 
        - vòng lặp thời gian trong 1 ngày : 0 -> 1 -> 2 -> ... -> 24h
        ví dụ tiếp theo vòng lặp các mùa 1 năm : xuân hạ thu đông . 

        vòng lặp for 
        for (expression 1; expression 2; expression 3) {
            // code block to be executed
        }
        việt hóa :
        for (biểu thức 1; biểu thức 2; biêu thức 3) {
            // code thực thi trong vòng lặp 
        }

        Expression 1 is executed (one time) before the execution of the code block.
        -> Biểu thức 1 được thực thi ( 1 lần ) trước khi đoạn code trong vòng lặp for được chạy

        Expression 2 defines the condition for executing the code block.
        -> biểu thứ 2 định nghĩa điều kiện vòng lặp được chạy

        Expression 3 is executed (every time) after the code block has been executed.
        -> biểu thức 3 được thực thi ( mỗi lần chạy vòng lặp) sau khi đoạn code trong vòng lặp được chạy xong
